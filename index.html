Phone Intellect

This project analyzes mobile service plans to help users find the best options based on data allowance and pricing. Using Java, Selenium WebDriver, Jsoup, React, JavaScript we developed web scraping tools to collect detailed data from Rogers, Bell, and Freedom, including plan names, costs, data allowances, and more. The data is then analyzed to recommend the most cost-effective plans. Enhanced search features include word completion and spell check. A Java-based web crawler processes content with frequency counting, inverted indexing, and page ranking to optimize search results and facilitate easy comparison of mobile plans.

Urban Forge

We developed an integrated security model designed to enhance store protection during periods of unrest using advanced machine learning algorithms. This system is capable of detecting abnormal situations, such as riots, and responds by automatically closing store doors to ensure safety. The model features multi-sensory data integration, combining sound, images, and live video captured by cameras and traffic lights for real-time analysis and decision-making. Additionally, it includes a dynamic alert system that deploys blinking traffic lights to signal unrest, providing immediate visual alerts to both authorities and the public. This project, executed by our team, leverages cutting-edge technology to effectively address security challenges and improve store safety during critical situations.
